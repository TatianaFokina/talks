  • Not all articles are translated into Russian language, and this is true for other languages as well
  • Sometimes, it's just a copy and paste from an official documentation without further or clear explanation
  • In many articles, there are no code demos
  • Contributing is not so easy
  • There can never be too many resources about web technologies ü§ì

How To Start

  1. Find Your Favourite Issue: Choose an issue from the project's issue tracker that interests you the most
  2. Fork the Repository with Content: Create a personal copy of the repository
  3. Write Your First Material: After forking the repository, you can start writing or editing content
  4. Create a Pull Request: Once your material is ready, you submit a pull request. Editors will help you to improve your article
